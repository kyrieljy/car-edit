# AI Mod Studio Prompt Batch Results V1

Last condensed: 2026-05-29 Asia/Shanghai

This is an archived summary of old manual prompt batch results. The previous long notes were condensed to avoid bloating new handoff context.

## Durable Findings

- Canvas lock can pass when prompt and references are disciplined.
- Multi-reference part images are useful, but they must not donate donor vehicle/background/wheels/paint.
- Hood body-color policy improved after explicit carbon handling.
- Product-only hood references are often weak for exact contour transfer; installed-angle references are better.
- Rear lip/spoiler tests require source images where the trunk deck is actually visible.
- Complex backgrounds are useful for preservation stress tests but should not be first-round precision tests.

## Historical Test Sources

- Source C: front/side parking-garage blue M4, useful for hood/body-color/canvas preservation.
- Source D/E/F: rear or side/rear visible samples, useful for spoiler, diffuser, exhaust, tail light, side skirt, and background preservation checks.

## Current Practice

Use small manual tests only when necessary and only with user approval for real provider calls. Record new failures as concise bad cases instead of appending long chronological logs here.
